1) Selectores (seleccionar elementos)

Regla de oro: si hay ID → usa getElementById. Con CSS usa querySelector/querySelectorAll.

Qué Sintaxis Devuelve
ID document.getElementById("menu") 1 elemento
Clase document.getElementsByClassName("item") HTMLCollection (viva)
Etiqueta document.getElementsByTagName("li") HTMLCollection (viva)
CSS (1) document.querySelector(".item") 1 elemento (el primero)
CSS (varios) document.querySelectorAll("li") NodeList (estática)
Truco CSS: etiqueta = "li" (sin símbolos), clase = ".item", id = "#menu".

Ejemplo

<ul id="menu">
  <li class="item">Inicio</li>
  <li class="item">Servicios</li>
</ul>

<script>
const menu = document.getElementById("menu");
const primeroItem = document.querySelector(".item");
const todosLosLi = document.querySelectorAll("#menu li");
</script>

2) children vs childNodes

children = solo etiquetas. childNodes = incluye textos (saltos/espacios) + etiquetas.

<div id="caja">
  <p>Hola</p>
  <p>DAW</p>
</div>

<script>
const caja = document.getElementById("caja");

console.log(caja.children.length);   // cuenta <p> (solo etiquetas)
console.log(caja.childNodes.length); // cuenta saltos/espacios + etiquetas
</script>
En HTML real suele haber saltos de línea → childNodes “ve” nodos texto. Por eso a veces “sale más”.

3) textContent (leer y modificar texto)

Regla: textContent es una propiedad (se usa con =).

<ul id="lista">
  <li>Manzana</li>
  <li>Plátano</li>
</ul>

<script>
const lista = document.getElementById("lista");

// Leer
console.log(lista.children[1].textContent); // "Plátano"

// Modificar
lista.children[1].textContent = "Pera";
</script>

4) Crear e insertar (createElement / appendChild / insertBefore)

Orden mental: crear → poner texto → insertar.

Crear y añadir al final

<ul id="lista">
  <li>A</li>
  <li>B</li>
</ul>

<script>
const lista = document.getElementById("lista");

const li = document.createElement("li");
li.textContent = "C";

lista.appendChild(li); // al final
</script>

Insertar en posición (antes de un nodo referencia)

<ul id="lista">
  <li>A</li>
  <li>C</li>
</ul>

<script>
const lista = document.getElementById("lista");

const li = document.createElement("li");
li.textContent = "B";

lista.insertBefore(li, lista.children[1]); // antes de "C"
</script>

5) Borrar (remove / removeChild)

Nemotecnia: remove() = “yo me borro”. removeChild() = “papá me borra”.

remove()

<script>
const lista = document.getElementById("lista");
lista.children[1].remove();
</script>

removeChild()

<script>
const lista = document.getElementById("lista");
lista.removeChild(lista.children[1]);
</script>

6) Mover vs copiar (appendChild vs cloneNode)

Regla MAESTRA: appendChild MUEVE. cloneNode(true) COPIA.

Acción Método Qué pasa
Mover destino.appendChild(nodo) Sale del origen y aparece en destino
Copiar const c = nodo.cloneNode(true) Original intacto + copia nueva

Ejemplo: copiar 1 nodo sin que desaparezca

<ul id="origen">
  <li>JS</li>
</ul>
<ul id="destino"></ul>

<script>
const origen = document.getElementById("origen");
const destino = document.getElementById("destino");

const liOriginal = origen.children[0];
const liCopia = liOriginal.cloneNode(true);

destino.appendChild(liCopia);
</script>

7) Listas vivas vs estáticas + bucles correctos

VIVA: cambia al mover nodos (HTMLCollection: children, getElementsByClassName, getElementsByTagName).
ESTÁTICA: no cambia (NodeList: querySelectorAll).

7.1 Lista VIVA → recorrer con while (mover)

<ul id="origen">
  <li class="item">A</li>
  <li class="item">B</li>
  <li class="item">C</li>
</ul>
<ul id="destino"></ul>

<script>
const destino = document.getElementById("destino");
const items = document.getElementsByClassName("item"); // VIVA

while (items.length > 0) {
  destino.appendChild(items[0]); // MUEVE
}
</script>

7.2 Lista ESTÁTICA → recorrer con for (copiar)

<ul id="origen">
  <li>A</li>
  <li>B</li>
  <li>C</li>
</ul>
<ul id="destino"></ul>

<script>
const origen = document.getElementById("origen");
const destino = document.getElementById("destino");

const items = origen.querySelectorAll("li"); // ESTÁTICA

for (let i = 0; i < items.length; i++) {
  const copia = items[i].cloneNode(true); // COPIA
  destino.appendChild(copia);
}
</script>
Tabla mental: lista viva → while. lista estática → for/forEach.

8) Copiar nodos específicos (por posición o por texto)

Flujo: seleccionar todos → recorrer → condición → clonar → añadir.

8.1 Copiar posiciones pares (0,2,4...)

<script>
const origen = document.getElementById("origen");
const destino = document.getElementById("destino");

const items = origen.querySelectorAll("li");

for (let i = 0; i < items.length; i++) {
  if (i % 2 === 0) {
    destino.appendChild(items[i].cloneNode(true));
  }
}
</script>

8.2 Copiar solo el que tenga texto "C"

<script>
const origen = document.getElementById("origen");
const destino = document.getElementById("destino");

const items = origen.querySelectorAll("li");

for (let i = 0; i < items.length; i++) {
  if (items[i].textContent === "C") {
    destino.appendChild(items[i].cloneNode(true));
  }
}
</script>
Ojo: comparar texto con ===. Y recuerda: "C" no es lo mismo que "c".

Quiero… Uso… Bucle
